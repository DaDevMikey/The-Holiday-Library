Quick Start

// Add to your HTML <script src="https://dadevmikey.github.io/The-Holiday-Library/ChristmasOverlay.js"></script> // Enable snow effect christmasOverlaySnow.enable(); // Or customize it christmasOverlaySnow.enable({ snowflakeCount: 100, snowflakeColor: 'white', minSize: 3, maxSize: 12, minOpacity: 0.4, maxOpacity: 1, zIndex: 9999 }); // Disable snow christmasOverlaySnow.disable();

API Options

Basic Usage

christmasOverlaySnow.enable(); christmasOverlaySnow.disable();

Custom Options

christmasOverlaySnow.enable({ snowflakeCount: 100, snowflakeColor: 'white', minSize: 3, maxSize: 12 });

Time-Based Scheduling

christmasOverlaySnow.enable({ startTime: '18:00', endTime: '23:59' });

Check Status

// Check if snow is active christmasOverlaySnow.isActive();
